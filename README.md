# Estrutura de Dados 1

Este repositório é destinado para as atividades, códigos e estudos referentes à disciplina Estrutura de Dados 1. Tal disciplina é cursada em 2026/01 na Universidade Tecnológica Federal do Paraná (UTFPR) do campus de Cornélio Procópio.

Principal referência bibliográfica: GOODRICH, Michael T.; TAMASSIA, Roberto. Estruturas de dados & algoritmos em Java. Bookman Editora, 2013.
